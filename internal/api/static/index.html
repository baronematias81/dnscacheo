Cifrado DNS
Estado y configuración de DNS over TLS (DoT) y DNS over HTTPS (DoH)
DNS over TLS (DoT)
RFC 7858 — Puerto 853/TCP
—
ProtocoloTCP con TLS — IPv4 + IPv6
Puertos853/tcp (IPv4) · [::]:853 (IPv6)
Configdot.enabled: true
CompatibilidadAndroid 9+, systemd-resolved, Unbound
DNS over HTTPS (DoH)
RFC 8484 — Puerto 443/TCP
—
ProtocoloHTTPS — IPv4 + IPv6
Método GETGET /dns-query?dns=<base64url>
Método POSTPOST /dns-query (wire format)
Configdoh.enabled: true
CompatibilidadFirefox, Chrome, iOS 14+, Windows 11
Configuración en config.yaml
server:
  ipv6_enabled: true              # habilita listeners IPv6 en puerto 53
  listen_udp6: "[::]:53"
  listen_tcp6: "[::]:53"
  upstreams6:                     # upstreams IPv6 (Cloudflare, Google, Quad9)
    - "[2606:4700:4700::1111]:53"
    - "[2001:4860:4860::8888]:53"
    - "[2620:fe::fe]:53"

tls:
  enabled: true
  auto_generate: true             # genera cert autofirmado si no existe
  min_version: "1.2"
  sans:
    - "dns.grupobaron.com.ar"
    - "192.168.1.1"
    - "2001:db8::1"               # IP pública IPv6 en el cert

dot:
  enabled: true
  listen: "0.0.0.0:853"
  ipv6_enabled: true
  listen_ipv6: "[::]:853"

doh:
  enabled: true
  listen: "0.0.0.0:443"
  ipv6_enabled: true
  listen_ipv6: "[::]:443"
  path: "/dns-query"
Configurar en Android
  1. Ajustes → Red e Internet → DNS privado
  2. Seleccionar "Nombre de host del proveedor de DNS privado"
  3. Ingresar: dns.tuservidor.com
Configurar en Firefox
  1. Preferencias → General → Configuración de red
  2. Activar "DNS sobre HTTPS"
  3. URL personalizada: https://dns.tuservidor.com/dns-query
